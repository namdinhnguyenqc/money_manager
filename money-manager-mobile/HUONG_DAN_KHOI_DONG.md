# 🚀 Hướng Dẫn Khởi Động Lại Server (Dành cho Sếp)

Nếu sếp thấy app bị mất kết nối (màn hình trắng hoặc báo không tìm thấy server), sếp chỉ cần làm theo các bước sau:

## 1. Cách Khởi Động Nhanh Nhất
Sếp mở Terminal (Dấu cộng ở thanh dưới hoặc nhấn `Ctrl + ~`) và gõ lệnh sau:

```bash
npx expo start --clear
```

*   **--clear**: Để xóa mọi cache cũ, giúp app chạy ổn định nhất.
*   **npx expo start**: Để khởi tạo lại máy chủ cho app.

## 2. Các Lỗi Thường Gặp & Cách Xử Lý
- **Lỗi Port 8081 already in use**: Sếp gõ `npx expo start --port 8082` (đổi sang cổng khác).
- **Lỗi Tunnel không chạy**: Sếp gõ `npx expo start --tunnel`.

## 3. Nhờ Em Hỗ Trợ
                                                                                                                                                                                                                                                                                                                                                                                                Nếu sếp lười gõ lệnh, sếp cứ nhắn tin cho em: **"Hết mạng rồi, bật lại server đi"** là em sẽ tự động quét sạch lỗi và bật lại server cho sếp ngay lập tức ạ! 🫡🔥
